# PHP

##OS	
+ CentOS 5.5 以上
+ Ubuntu 11 以上
+ RedHat  Enterprise Linux (RHEL) 5 or later

##PHP版本	
+ PHP 5.2.13 及以上
+ PHP 5.3 及以上
+ PHP 5.4 及以上
+ PHP 5.5 及以上
+ PHP 5.6 及以上

##数据库	
mysql [mysql , pdo , mysqli]

##webServer	
+ Apache 2.2 或 2.4 mod_php 模式
(线程安全与非线程安全)
+ 任何用 通过 FastCGI 连接 php-fpm (如 Nginx)

##明确不支持的框架	
Yaf

##可以安装的框架	
所有基于PHP语言写的框架都可以安装 PHP-Agent
